# qalam_cli
A simple cli for qalam

To build for your system you need the following libraries:
curl and its dependencies.
cpr. https://github.com/libcpr/cpr.
gumbo-parser.https://github.com/google/gumbo-parser
gumbo-query. https://github.com/lazytiger/gumbo-query

You will need the main.cpp and everything in src. You can use the build.py / mkfile2 to build the project as well
